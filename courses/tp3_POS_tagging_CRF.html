Tools

  • CRF++: enable you to create and use CRF models.
  • Installation:
    • Download the installation script CRF_installation.bash then type:
      bash CRF_installation.bash
      WARNING: the tool will be installed by default into the "tools" directory located into your home directory.

    • Once the installation is done, type 
      source ~/.bashrc
    • Test your installation by typing 
      crf_learn --help
      the result should print the help of the CRF command.

Data

  • Use the same data files given in the Demo 2: train.words.txt, test.words.txt, train.tags.txt and test.tags.txt available here
  • Decompress then using tar:
    tar xvfz tp3_files.tgz
    or 
    gunzip tp3_files.tgz
    and then 
    tar xvf tp3_files.tar
